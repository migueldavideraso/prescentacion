# Enlaces con Códigos QR

Esta aplicación muestra una serie de enlaces importantes junto con sus códigos QR correspondientes.

## Enlaces incluidos

- [Lovable.dev](https://lovable.dev/)
- [v0.app](https://v0.app/)
- [Claude Code](https://www.claude.com/product/claude-code)
- [Google Code Assist](https://codeassist.google/)
- [Windsurf Editor](https://windsurf.com/editor)
- [Google Antigravity](https://antigravity.google/)
- [Trae AI](https://www.trae.ai/)

## Instrucciones para ejecutar el proyecto

1. Instalar dependencias:
   ```bash
   bun install
   ```

2. Iniciar el servidor de desarrollo:
   ```bash
   bun run dev
   ```

3. Para construir el proyecto para producción:
   ```bash
   bun run build
   ```

## Tecnologías utilizadas

- Vite
- Svelte
- QRCode.js (para generar los códigos QR)
- Bun (como gestor de paquetes)
